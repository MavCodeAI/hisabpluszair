## Quick Start

### Prerequisites
- Flutter SDK (3.0+)
- Android Studio / VS Code
- Device or emulator

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/hisaabplus-flutter-app.git
   cd hisaabplus-flutter-app
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
